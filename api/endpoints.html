Base URL

All endpoints are relative to: http://localhost:8000

Endpoints

Get All Fun Facts

GET/api/funfact

Returns an array of all available fun facts.

Response Format

[{
    "id": 1,
    "fact": "string",
    "author": "string"
}]

Example Response

[{
    "id": 1,
    "fact": "Honey never spoils. Archaeologists have found 3000-year-old honey and it was still good.",
    "author": "Science Daily"
},
{
    "id": 2,
    "fact": "Octopuses have three hearts and blue blood.",
    "author": "National Geographic"
}]

Response Codes

  • 200 - Success
  • 404 - Not Found
  • 500 - Server Error

Rate Limiting

Currently, there are no rate limits implemented on the API.

CORS

This API supports Cross-Origin Resource Sharing (CORS) and can be accessed from any origin.
